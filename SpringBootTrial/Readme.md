# プロジェクト名
SpringBootTrial

# 概要
Spring Bootを用いたWebアプリケーションを試しに作っていくプロジェクト

# 開発環境

## 基本環境
* Windows11 64bit
* eclipse 2022-03
* Java Open JDK 11
* Spring Boot Framework

## eclipse上、プロジェクト設定
* Type
  * Maven Project
* Packaging
  * War
* Java Version
  * 11
* Language 
  * Java

## 外部ライブラリ
* jQuery
* Semantic UI
